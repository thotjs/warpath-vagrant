### Warpath Vagrant

This repo holds the various vagrant files for managing our coreos servers in different environments

Usage:

``` bash
  git clone git@github.com:thotjs/warpath-vagrant.git

  cd {environment}

  vagrant up
```

The possible environments are:
* local
* dev
* staging
* production
